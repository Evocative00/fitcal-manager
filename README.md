# FitCal Manager

개인 맞춤형 **식단 및 칼로리 관리 매니저** 프로젝트입니다.

사용자의 신체 정보와 목표를 입력받아 하루 권장 칼로리와 탄단지 권장량을 계산하고, 이후 식단 기록과 대시보드 기능으로 확장할 예정입니다.

---

## 기술 스택

### Frontend

- React
- Vite
- JavaScript
- Tailwind CSS
- Axios
- React Router
- Recharts

### Backend

- Java 17
- Spring Boot
- Spring Data JPA
- Validation
- Lombok
- MySQL Connector

### Database

- MySQL 8.4
- Docker Container로 실행

### DevOps / Environment

- Docker
- Docker Compose
- GitHub
